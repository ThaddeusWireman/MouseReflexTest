Mouse Reflex Test User Manual

When the user hits play, an introduction screen will appear asking for the user to enter a username. An error will be shown if the user attempts to submit an empty username box or a username that is longer than 12 characters in length. The user is then prompted to choose their difficulty selection by clicking the normal or hard buttons. Once a diffulty has been selected the game will start immediately. Red buttons will be displayed at random locations on the screen for 30 seconds and the amount of buttons clicked during the 30 seconds are displayed. When the 30 second timer is up, the user's number of buttons pressed and the average time for each button pressed is displayed along with a Reset? button that brings the user back to the original introduction screen. If the user does not click any targets, a large error message explaining that an average can not be calculated with a score of 0.
